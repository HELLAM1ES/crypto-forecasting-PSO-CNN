Crypto Data API

Projet réalisé dans le cadre du master MoSEF - Data Science

Objectifs de l'API

Cette API permet d'extraire, traiter et analyser des données de crypto-monnaies pour :

  • Prédire les rendements des crypto-monnaies (return prediction).
  • Prédire la direction du marché (direction prediction).

Description des Endpoints

  • Scrape Data: Extrait des données de marché pour les cryptos.
  • Merge and Features: Fusionne les données et génère des indicateurs techniques.
  • Train Return Model: Entraîne un modèle pour prédire les rendements des cryptos.
  • Train Direction Model: Entraîne un modèle pour prédire la direction du marché.
  • Predict Direction: Fournit une prédiction de la direction pour une ou plusieurs cryptos.
  • Calculate MVF: Calcule les métriques financières (rendement attendu, volatilité, etc.).
